# 💰 Expense Tracker (Python CLI)

A simple **Command-Line Expense Tracker** built using **Python**. This project was created while learning Python fundamentals and helped me practice core programming concepts by building a real-world mini application.

---

## 📌 Features

- ➕ Add new expenses
- 📋 View all recorded expenses
- 💵 Calculate total spending
- 🗂️ Store expense details using Python dictionaries
- 🖥️ Interactive menu-driven interface

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/VedikaPP/expense-tracker-python.git
```

### Go to the project folder

```bash
cd expense-tracker-python
```

### Run the application

```bash
python expense_tracker.py
```
